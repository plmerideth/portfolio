  • Transforms let you translate, rotate, scale and skew any element.
  • Translation functions allow you to move elements left, right, up, or down
  • transform: translate(45px, -45px); You can also do translateX() or translateY() for vertical or horizontal movement only
  • Transforms don’t work on inline elements. But that’s easy enough to fix. Just add display: inline-block; to the span:
  • The scale(x,y) function scales an element by the defined factors horizontally then vertically. If only one value is provided, it will be used for both the x and y values
  • To declare multiple transformations, provide a space-separated list of transform functions.

  • The skew(x,y) function specifies a skew along the x and y axes
  • Transitions allow the values of CSS properties to change over time, essentially providing simple animations.
  • CSS animations, unlike transitions, allow you to control each step of an animation via keyframes.
  • A keyframe is a snapshot that defines a starting or end point of any smooth transition
