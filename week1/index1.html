Mobile UX Video

  • Very interesting videos. Most interesting points:

    • - Always been taught mobile first design approach

    - 75% of mobile users use thumbs

    - 49% use one hand

    - Reduce download time with .svg, webfonts, minify code, and GZIP to compress data

    - Put nav bar at bottom of screen for easier one-handed access

    - MENU with a border gets 20% more clicks

    - Single line credit card input was very nifty!

    - Insert labels in input fields to save real estate

    - Don't use pop-ups and pop-overs!
